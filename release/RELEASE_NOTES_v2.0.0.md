# GeminiMate v2.0.0 发布说明

发布日期：2026-03-13

## 核心更新
- 重构并细化了 README，完整覆盖控制面板功能（主面板 + 设置面板）。
- 时间线定位逻辑修复：
  - 移除会误命中左侧区域的锚点回退规则。
  - 仅接受右侧锚点；锚点不可用时回落到右侧默认定位。
- 控制面板信息架构优化：
  - 思维链翻译开关与思维链模式保持连续，不再被“视频推荐屏蔽”割裂。
- 导出入口调整：
  - 弹窗中的导出 UI 已移除。
  - 内容脚本不再启动导出按钮注入。
- LaTeX 标记策略优化：
  - 移除 `Fixed by GeminiMate` 文案提示。
  - 仅对“真实修复”的公式显示红色提示线。
- Markdown 修复路径优化：
  - 从块级 `innerHTML` 重写改为文本节点级修复，降低链接/交互节点失活风险。
