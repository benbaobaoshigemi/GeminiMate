# GeminiMate

GeminiMate 是一款专为重度依赖 Google Gemini 的研究人员、开发者及知识工作者打造的全能增强型浏览器扩展。

它从重构页面排版入手，深入干预复杂的公式解析、格式修正与网络请求，并内嵌了针对 Gemini 原生痛点的诸多“杀手锏”级工具箱。不仅优化了视觉体验，更实质性地将网页端重塑为一个高度可定制化的高效生产力工作台。

---

## 核心功能清单

GeminiMate 的能力矩阵涵盖以下七大核心领域：

### 1. 核心修复与内容翻译（解决原生核心顽疾）
Gemini 与其网页端的配合常常存在断层，容易导致乱码与阅读障碍，GeminiMate 在这部分进行了自研深挖：
*   **LaTeX 完美修复**：Gemini 生成长篇公式常出现未闭合或结构错乱，让人看着极其恼火。内置自研的 DOM 快照全景修复引擎，能够完美修复这些乱码与出界，并智能避开系统底层的 思维链（Thought）渲染区防止误伤。
*   **Markdown 断裂缝合**：完美解决流式插入时常出现的“加粗（`**`）样式突然断裂与失效”问题，填补渲染裂缝。
*   **思维链（Thought）原生翻译**：针对长篇难以阅读的原版英文思考树，自带处理并提供直观的原生翻译对比与替换，使推理链路一目了然。

### 2. UI 调整与沉浸排版（将阅读控制权交还用户）
*   **全局三段式宽度调节**：允许完全自由拖拽并调整「对话区域」、「输入框」和「侧边栏」的宽度，释放你的高分宽屏空间。（*灵感参考自 Voyager*）
*   **极致排版控制**：允许自由调整字体大小、行距选项与字间距，更支持**自行导入系统内的本地字体**直接应用。
*   **去除视觉干扰**：一键去除页面底部的无效官方说明与对话遮罩，同时提供拦截开关以移除刺眼的 AI Ultra 升级广告。
*   **首行缩进引入**：支持开启段落首行缩进，更契合传统的中文长文阅读习惯。
*   **强调文本样式切换**：支持将所有重要强调文本在“**加粗**”和“<ins>下划线</ins>”之间随心切换，减轻全屏黑体带来的视觉疲劳。

### 3. 高频动作增强（快速交互）
*   **公式多轨拷贝**：单击页面数学公式即可一键复制。内置对 MathML 与原生 LaTeX 的双重支持处理。（*灵感参考自 Voyager*）
*   **悬浮引用提问**：用鼠标划选某段回复内容后激活附着按钮，可一键将该内容自动转化为规范引用块并转填入输入框，极其适合追问多轮长对话。（*灵感参考自 Voyager*）
*   **原图去水印引擎**：直接拦截图片下载通道或图层，去除生成图片时自带的底层与表层水印标识。（*灵感参考自 Voyager*）

### 4. 高级时间线与对话导航（在巨量对话中精准定位）
在继承初代树状导航概念的基础上做了全维度的重构与扩展：
*   **智能侧边时间线**：便捷定位各节点对话内容，并支持直接调整时间线的界面粗细。
*   **自动向上刷新拉取**：克服长对话网页节点流失不渲染的问题，一键溯源刷新完整对话。
*   **会话内局部检索**：在侧栏中直接对当前页面进行关键词检索，且**支持检索并高亮所有模型的“回复部分”**，并在鼠标悬停时展示对话详情面板。（*改良自 Voyager 并深度增强*）

### 5. 实时网络质量监测（把链路状态直接摊开）
*   **后台持续探测 Gemini 链路**：后台服务会持续对 `gemini.google.com` 发起探测，默认以前台 1 秒级采样为主，并在后台休眠恢复后自动补测，不依赖页面是否仍处于当前对话状态。
*   **左侧悬浮三指标胶囊**：在对话页左侧居中显示 `延迟 / 抖动 / 丢包率` 三枚独立胶囊，直接以颜色分段反映当前链路处于绿色、蓝色还是红色区间。
*   **设置页可调阈值**：在设置页提供三组“三色双点”阈值滑杆，允许分别定义各指标从绿色过渡到蓝色、再进入红色的分界点，并支持一键恢复默认值。

### 6. 知识归档与导出引擎 (重塑管理端)
*   **一键转录为高定版 Word (.docx)**：支持单条优质回复导出为含有精确数学矩阵公式的 Word 文档。区分普通与学术两种环境：**普通模式**完整保留网页字色环境；**学术模式**则将全盘剥离样式并匹配规范的学术界字体级论文排版呈现。
*   **会话局部保存为多格式**：支持提炼内容为标准 PDF、Markdown 乃至 JSON 源数据归档。（*灵感参考自 Voyager*）
*   **纯本地化文件夹重构**：侧边栏支持建立本地文件夹目录树系统保存对话树。注：在此版本中**已坚决移除并杜绝所有第三方云同步组件**，100% 捍卫个人隐私与本地数据安全。（*源于 Voyager 但重塑了隐私规范*）

### 7. 解耦渲染与外挂小工具 (解决极度受限情景)
*   **代码框图表即挂即渲**：
    *   支持 Mermaid 重新渲染（*重构了全新 UI 实现*），并在全新的沙箱内实现了原生的 **SVG 图表独立绘制支持**与 HTML 即时预览。
    *   支持所有渲染出的新图与代码随意互切查看，并支持统一**使用独立新网页**打开无痛全屏预览。
*   **带代码后缀下载识别**：对代码模块进行识别判定，点击下载后全部按照对应代码语言配置对应主流后缀文件格式。
*   **图像欺骗型 PDF 小工具箱**（专用于攻克 Gemini 输入限制）：
    *   **长图切割转 PDF**：解决 Gemini 视力极差、难以正确识别处理包含超长数据截屏图的问题。
    *   **多图聚合转 PDF**：直接将几十张图片拼合成单一文档抛给模型库，完美避过 Gemini 一次只能固定上传 10 张图片的死结。

## 致谢与独立溯源说明

GeminiMate 扩展在起步及初步验证阶段，曾极大受益并参考了开源项目 [gemini-voyager](https://github.com/upupming/gemini-voyager) 的部分界面理念与功能列表设计（如基础的时间线轮廓、基本的区块宽度拖拽干预、复制引用理念或多格式粗切导出等）。Voyager 为改善原生枯燥的体验提供了一个不可多得的拓荒范例。

但在项目快速延展的深水区开发中，GeminiMate 早已历了从代码到底层功能的彻底迭代与重做，完全确立了以独立技术栈驱动的路线：
1. **代码基盘异构**：扬弃了原有项目大一统的 `content.js` 胶水组织形式；重制为极度注重类型安全体验与业务解耦的 **TypeScript** 面向对象架构组合（可通过本地 `npm run typecheck` 等指令闭环校验）。
2. **重度痛点针对性研发**：在应对学术与排版的顽疾诉求上，GeminiMate 中包含 `带快照防干涉的 LaTeX/Markdown 修补引擎`、`包含洗版对比概念的双语翻译器`、`直接与标准 Word 公式流打通且包含双轨排版系统的单条文档中心`、`图表渲染沙箱页面机制`以及`规避限额策略的 PDF 切割机` 等大量重型高价值核心能力，这些特有版块均代表着 GeminiMate 全新而完全独立于参考系之外的产品原创力体现。
